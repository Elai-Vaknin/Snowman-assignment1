# Project #1 in CPP

Create a snowman ASCII art using 8 digits code.
Each digit represents a part in the snowman body with 4 options.

The structure of the Snowman:</br>
<center>
 HHHHH</br>
 HHHHH</br>
X(LNR)Y</br>
X(TTT)Y</br>
 (BBB)</br>
</center>

H is for hat.
X is for left arm.
Y is for right arm.
L is for left eye.
R is for right eye.
N is for nose.
T is for torso.
B is for base.
